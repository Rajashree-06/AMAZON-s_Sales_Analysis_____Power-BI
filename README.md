# AMAZON-s_Sales_Analysis_____Power-BI

📊 Amazon Sales Data Analysis – Power BI Project

This Power BI project focuses on analyzing Amazon’s combined dataset to uncover meaningful business insights.
The report helps understand sales performance, customer patterns, product profitability, and operational trends.

🔍 Key Features

Data Cleaning & Transformation: Handled using Power Query to remove duplicates, fix data types, and prepare the dataset.

Data Modeling: Relationships designed for efficient analysis.

DAX Measures: Created for KPIs like Total Sales, Total Orders, Profit, Ratings, Delivery Status, etc.

Interactive Dashboard:

Revenue & Order Trend Analysis

Top-Performing Products & Categories

Customer Ratings Breakdown

Fulfillment & Delivery Insights

Region-Wise Performance

🎯 Project Objective

To demonstrate the ability to clean, model, and visualize real-world e-commerce data and deliver insights useful for decision-making.

🛠 Tools Used

Power BI

Power Query

DAX

Excel (CSV dataset)
